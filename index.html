Цветовая палитра

Ниже представлен список основных цветов и их применение. Нажмите на название цвета для подробностей.

Главный Красный ▸

HEX: #D72638

Основной брендовый цвет. Используется для акцентирования главных элементов интерфейса и важных действий пользователя.

Ночной Синий ▸

HEX: #1B1B3A

Фоновый цвет для тёмных областей, придающий интерфейсу стиль Obsidian. Идеален для фона страниц и панелей.

Ярко-Оранжевый ▸

HEX: #FF9F1C

Вторичный акцентный цвет. Используется для привлечения внимания к вспомогательным действиям или важной информации.

Мятный ▸

HEX: #A8E6CF

Цвет для позитивных элементов (например, сообщений об успехе). Добавляет ощущение свежести и контраста.

Светлый Серый ▸

HEX: #F0F0F0

Нейтральный светлый цвет для фона светлых элементов, текста на тёмном фоне или разделительных линий.

Тёмный Серый ▸

HEX: #333333

Нейтральный тёмный цвет для текста на светлом фоне или второстепенного фона в тёмной теме.

Акцентный Голубой ▸

HEX: #3E92CC

Дополнительный акцентный цвет (например, для ссылок или выделения выбранных элементов). Хорошо сочетается с основными цветами.

Типографика

В дизайн-системе используются три шрифта: Montserrat для заголовков, Open Sans для основного текста и Pacifico для акцентных элементов.

Дополнительные рекомендации

  • Используйте единый стиль на всех страницах приложения.
  • Соблюдайте иерархию заголовков и отступов для удобства чтения.
  • Ограничивайте ширину контента для лучшей читаемости на больших экранах.
  • Придерживайтесь заданных цветов и шрифтов дизайн-системы для единообразия интерфейса.
